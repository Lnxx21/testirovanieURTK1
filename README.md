# testirovanieURTK1
Рицлер Влад, Елунин Влад
Написать программу на языке программирования C#, выполняющую четыре арифметические операции(+, -, *, /) двух чисел, введенных пользователем. Каждая операция должна быть в отдельном методе.

Пример для операции плюс:
ввод 1 переменной "15"
ввод 2 переменной "25"
выбор операции (+, -, *, /): +
вывод 40

ввод 1 переменной "14"
ввод 2 переменной "25"
выбор операции (+, -, *, /): +
вывод 39

Пример для операции минус:
ввод 1 переменной "15"
ввод 2 переменной "25"
выбор операции (+, -, *, /): -
вывод -10

ввод 1 переменной "14"
ввод 2 переменной "25"
выбор операции (+, -, *, /): -
вывод -11

Пример для операции деление:
ввод 1 переменной "30"
ввод 2 переменной "2"
выбор операции (+, -, *, /): /
вывод 15

ввод 1 переменной "50"
ввод 2 переменной "2"
выбор операции (+, -, *, /): /
вывод 25

Пример для операции умножение:
ввод 1 переменной "20"
ввод 2 переменной "2"
выбор операции (+, -, *, /): *
вывод 40

ввод 1 переменной "100"
ввод 2 переменной "2"
выбор операции (+, -, *, /): *
вывод 200
